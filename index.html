TUBO-QUEST PARKING

Controles

  • : avancer

  • : freiner/reculer

  • : tourner a gauche

  • : tourner a droite

Regles

  • placer votre voiture sur les places jaune

  • chaque place validé rapporte 10 points

  • une fois toutes les places d'une map validée vous passerez a la map suivante

  • validez toutes les places de toutes les map pour finir le jeu

  • chaque collision avec une autre voiture vous fera perdre 5 point
